Encabezado de main

La etiqueta main HTML5 sirve para definir el contenido principal o más destacado. Solo puede haber una etiqueta main.

La etiqueta footer contiene la agrupación de código HTML correspondiente al pie de página que suelen tener habitualmente todas las webs. El tipo de información en esta etiqueta HTML5 puede ser:

  • Información legal
  • Contactar
  • Privacidad
  • Otros enlaces de interes
